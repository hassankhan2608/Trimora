# Support

Trimora is a small open-source project maintained in spare time. Please pick the right channel for your question to keep things tidy.

## Where to get help

| You want to…                                  | Use this                                                                                       |
| --------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| Read setup or API docs                        | The [README](README.md)                                                                        |
| Report a bug                                  | [Open an issue](https://github.com/hassankhan2608/Trimora/issues/new) with the **bug** label   |
| Request a feature                             | [Open an issue](https://github.com/hassankhan2608/Trimora/issues/new) with the **feature** label — please first check the *Project scope* section in [CONTRIBUTING.md](CONTRIBUTING.md) |
| Ask a usage question                          | [Open a discussion](https://github.com/hassankhan2608/Trimora/discussions) (or an issue if discussions are disabled) |
| Contribute code                               | See [CONTRIBUTING.md](CONTRIBUTING.md)                                                         |
| Report a security vulnerability               | See [SECURITY.md](SECURITY.md) — **do not open a public issue**                                |

## Before opening an issue

Please check that:

1. You're on the latest commit of `main`.
2. Your `.env` matches `.env.example` (no missing or misspelled variables).
3. You've checked the API logs (`docker compose logs api`) and browser console for errors.
4. Your question or bug isn't already in [open issues](https://github.com/hassankhan2608/Trimora/issues?q=is%3Aissue) or [closed issues](https://github.com/hassankhan2608/Trimora/issues?q=is%3Aissue+is%3Aclosed).

## Response times

This is a hobby project — there are **no SLAs**. Maintainers reply when they can, usually within a few days. Polite pings on a stale issue are fine.

## What is *not* supported

- Help deploying behind specific cloud providers, reverse proxies, or auth systems beyond what's documented in the README.
- Custom features outside of [project scope](CONTRIBUTING.md#project-scope). You're welcome to fork.
- Anything related to commercial / paid use — Trimora is MIT-licensed; use it however you like, but support is community-driven.

Thanks for using Trimora.
